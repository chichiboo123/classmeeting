# 🏫 우리반 학급회의

> 초등학교 학급회의를 위한 웹 기반 진행 도우미

[![GitHub Pages](https://img.shields.io/badge/Live-chichiboo123.github.io-2C3E7A?style=flat-square)](https://chichiboo123.github.io/classmeeting)

---

## 개요

학급회의의 6단계 순서를 안내하고, 사회자 대본 확인 / 의견 기록 / 결과 저장을 한 화면에서 처리할 수 있는 교사·학생 공용 도우미 앱입니다.
별도 설치 없이 브라우저에서 바로 사용할 수 있으며, 단일 HTML 파일로 구성되어 있습니다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| **기록 모드** | 단계별 사회자 대본(접기/펼치기) + 의견·메모 입력 |
| **발표 모드** | 핵심 안내를 PPT 슬라이드처럼 큰 화면으로 표시 |
| **목표 연동** | STEP 3에서 입력한 이달의 목표를 발표 모드에서도 표시·편집 |
| **자동 저장** | 작성한 회의 내용을 브라우저에 자동 저장 — **새로고침해도 유지** |
| **내용 초기화** | 작성한 회의 내용을 한 번에 모두 지우고 새 회의 시작 |
| **결과 저장** | 회의 기록을 클립보드 복사 / TXT / JSON으로 저장 |
| **기록 복원** | 저장한 JSON 파일을 불러와 이전 회의 내용 복원 |
| **텍스트 편집** | 우리반 약속·대본·단계 제목 등 화면 문구를 직접 수정 (인라인 편집 + 콘텐츠 설정) |
| **콘텐츠 백업** | 수정한 문구 설정을 JSON 파일로 내보내기·불러오기 (기기 간 이동) |

## 학급회의 진행 순서

```
STEP 1  개회선언 & 우리반 약속 선서   🟠
STEP 2  거울(돌아보기) 회의           🟢
STEP 3  다음 달 목표 정하기           🟣
STEP 4  제안하기                      🟡
STEP 5  선생님 말씀                   🩷
STEP 6  폐회 선언 & 결과 저장         🟢
```

---

## 사용 방법

### 기록 모드 (기본)
1. 상단 **회의 월** 입력란에 해당 월을 입력합니다.
2. 각 단계를 순서대로 진행하며 의견과 메모를 기록합니다.
3. 사회자 대본은 탭을 눌러 펼쳐 확인할 수 있습니다.
4. STEP 6에서 회의 결과를 저장합니다.

### 발표 모드
1. 우측 상단 **발표모드** 버튼을 클릭합니다.
2. 각 단계의 핵심 안내가 슬라이드 형태로 표시됩니다.
3. **다음 →** 버튼으로 단계를 이동합니다.
4. STEP 1은 개회선언 / 우리반 약속 선서 2페이지로 구성됩니다.
5. STEP 3 발표 화면 하단에서 목표 텍스트를 바로 입력·수정할 수 있습니다.

## 데이터 저장 & 초기화

작성한 회의 내용(회의 월, 메모, 거울회의 기록, 의견·제안, 목표, 선생님 말씀)은
입력하는 즉시 브라우저(localStorage)에 **자동 저장**됩니다. 따라서 실수로 새로고침하거나
탭을 닫았다가 다시 열어도 작성 내용이 그대로 유지됩니다.

- **초기화** — STEP 6 하단 **「작성 내용 모두 지우기」** 버튼으로 모든 회의 내용을 한 번에 비우고
  새 회의를 시작할 수 있습니다. (편집한 문구·콘텐츠 설정은 유지됩니다.)
- **영구 보관** — 장기 보관이나 공유가 필요하면 STEP 6에서 **TXT / JSON 내보내기**를 이용하세요.
  localStorage는 같은 브라우저에서만 유지되며, 브라우저 기록 삭제 시 함께 지워집니다.

> 저장 항목은 ① 회의 내용(`classmeeting_record_v1`)과 ② 편집한 화면 문구(`classmeeting_content_v1`)로
> 분리되어 있어, 회의 내용을 초기화해도 수정해 둔 문구는 그대로 유지됩니다.

## 텍스트(문구) 편집

앱에 표시되는 **우리반 약속, 사회자 대본, 단계 제목, 폐회 인사** 등 핵심 문구를 직접 고칠 수 있습니다.

- **인라인 편집** — 상단 ✏️(편집) 버튼을 켜면 화면의 점선 영역을 클릭해 바로 수정합니다.
  같은 문구(우리반 약속·단계 제목)는 여러 곳에 있어도 한 번에 함께 바뀝니다.
- **콘텐츠 설정** — 상단 ⚙️(설정) 버튼을 누르면 모든 문구를 한곳에서 수정하거나,
  항목별·전체를 기본값으로 되돌릴 수 있습니다.
- **백업** — 콘텐츠 설정 창에서 수정한 문구를 **JSON 파일로 내보내기 / 불러오기** 할 수 있어
  다른 기기나 동료 교사와 손쉽게 공유할 수 있습니다.
- 문구 속 `{월}` 표시는 입력한 회의 월로 자동 치환됩니다.
