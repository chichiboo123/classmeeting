사용법
우리반 학급회의 도우미
edit_note
기록 모드각 단계를 순서대로 진행하며 회의 내용을 입력해요. 사회자 대본은 탭을 눌러 펼칠 수 있어요.
slideshow
발표 모드단계별 핵심 안내를 슬라이드 형식으로 보여줘요. 화면에 띄워 학급 전체가 볼 수 있게 해요.
swap_horiz
단계 이동하단 '다음 →' 버튼 또는 상단 진행 탭을 눌러 단계를 이동할 수 있어요.
push_pin
목표 연동STEP 3에서 입력한 목표는 발표 모드에서도 표시·수정할 수 있어요.
ios_share
결과 저장STEP 6에서 회의 결과를 클립보드·TXT·JSON으로 저장하거나 JSON 파일로 불러올 수 있어요.
edit
텍스트 편집상단 edit 버튼으로 편집 모드를 켜면 우리반 약속·대본 등을 화면에서 바로 고칠 수 있어요. tune 버튼을 누르면 모든 문구를 한곳에서 수정하거나 백업 파일로 내보내기·불러오기 할 수 있어요. 수정 내용은 자동 저장돼요.
Created by. 교육뮤지컬 꿈꾸는 치수쌤
콘텐츠 설정
앱에 표시되는 문구를 수정해요 · 변경 즉시 저장
수정 내용은 이 브라우저에 자동 저장돼요. 다른 기기로 옮기려면 「설정 내보내기 / 불러오기」를 이용하세요.
edit 편집 모드 · 점선 영역을 클릭해 텍스트를 바로 수정하세요. 「{월}」은 회의 월로 자동 표시돼요.
